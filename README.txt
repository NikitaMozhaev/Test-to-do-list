Инструкция для запуска web-приложения "Список Задач".
1. Для запуска нужны программы MySql, XAMPP.
2. Поместить всю папку проекта в C:\xampp\htdocs
3. Поместить папку to_do_list в C:\xampp\mysql\data
4. Открыть XAMPP Control Panel и включить Apache и MySQL
5. В браузере перейти по адрессу http://localhost/to-do-list/index.php.
